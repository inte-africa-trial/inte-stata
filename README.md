# inte-stata
STATA do files


copy the `get_env.do` to ~/path/to/dta/folder

.. code-block:: bash

    cp get_env.do ~/path/to/dta/folder

create an `env.txt`

.. code-block:: bash

    ~/path/to/do/folder
    ~/path/to/dta/folder
    yyyymmdd


where `yyyymmdd` is the timestamp suffix on the dta files.

copy the `env.txt` to ~/path/to/dta/folder

open STATA:

.. code-block:: bash

    cd ~/path/to/dta/folder
